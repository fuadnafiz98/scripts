## Initial setup

Make sure you make the `install.sh` file executable.

```bash
sudo chmod +x install.sh
```

Run `install.sh` to install all the scripts. By default scripts scripts will be installed in `/usr/bin/`

**Install all the tools**

```bash
./install.sh
```
